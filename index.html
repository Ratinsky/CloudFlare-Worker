Notion / Charts

Free charts
inside Notion

Embed unlimited charts into any Notion page via /embed block.
No backend, no server — just one HTML file on GitHub Pages.

STEP 01
Create a Notion Integration
Go to notion.so/my-integrations, create an integration, copy the Internal Integration Token.

Then open your Notion database → ⋯ → Connections → add your integration.
STEP 02
Deploy to GitHub Pages
Create a GitHub repo, put this index.html there, enable GitHub Pages in Settings → Pages → Deploy from branch.
STEP 03
Configure the CORS proxy
Notion API doesn't allow browser requests directly. Deploy a free proxy worker on Cloudflare Workers (free tier: 100k req/day):
// Cloudflare Worker code: export default { async fetch(request) { const url = new URL(request.url); const target = url.searchParams.get("url"); if (!target) return new Response("missing url", { status: 400 }); const res = await fetch(target, { method: request.method, headers: { "Notion-Version": "2022-06-28", "Content-Type": "application/json", }, body: request.method !== "GET" ? request.body : undefined, }); const data = await res.text(); return new Response(data, { headers: { "Content-Type": "application/json", "Access-Control-Allow-Origin": "*", "Access-Control-Allow-Headers": "*", } }); } };
After deploy, put your Worker URL into the PROXY variable at the top of index.html.
STEP 04
Embed into Notion
In Notion type /embed and paste your URL with parameters:
https://yourname.github.io/notion-charts/?token=secret_xxx &db=DATABASE_ID &type=bar &x=Name &y=Revenue &title=Monthly+Revenue
All URL parameters:
token
Notion Integration Token
secret_xxx...
db
Database ID from URL
32-char string
type
Chart type
bar · line · pie · doughnut · scatter
x
Property for X axis / labels
column name
y
Property for Y axis / values
column name
title
Chart title (optional)
any text
limit
Max rows (optional)
default: 100
sort
Sort direction (optional)
asc · desc
